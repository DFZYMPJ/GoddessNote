<h1>使用wxPython打造自己的笔记本</h1>
<quo>为什么打造一款这样的笔记本？</quo>
<ul><li>笔记需要云同步，有点担心隐私泄露。</li>
<li>如果你也想拥有个笔记本应用，那么clone源码到本地运行它。</li>
</ul>
